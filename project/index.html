Site Improvements



Responsive

We've created a mobile-friendly design that adapts to tablets, different browsers, and phones to make it easier than ever for students to access class materials from anywhere, whenever they need them.

Updated Code

We've added a jQuery confirmation pop up upon submitting quiz. We've also added scroll to top animation which helps students to find a topic easier.

HTML5

We've upgraded the site from XHTML 1.0 to HTML5, the most current and flexible iteration of HTML code. This will result in a better user experience and usability.




CSS3

We've used CSS3 to style our pages. By using whitespace, headings and two different background colors, our design is trying to make scanning and reading the page content easier for students.

Search Engine Optimization

We've added meta content to increase the searchability and favourable ranking of the site pages so they come up more quickly should a student be searching for them.

Included Files

We`ve included header, navigation and footer to reduce duplicate code. Reducing duplicate code makes our website faster.



Features For Future



Forums

Online Forums create a way for students to discuss about course and assignments.It also provides a place for students to get answers to questions from other students and their instructor.

We are planning to integrate Comp 1950 page with Learning Hub web site to make it work as one platform for all students without need to go for an information. Integration will help to check if the student is allowed to veiw the page or submit the quiz based on the student ID and this way to do a proper validation.

Live Example
